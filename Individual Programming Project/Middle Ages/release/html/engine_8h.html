Opis szczegółowy

Interface of game engine.

Contains all functions needed to start, play, read input (and check parameters), print output and end the game.

Initializes a game with size of a board, number of rounds and positions of kings.

One INIT for each player is needed before any other Command.

Parametry
[in]nSize of a board.
[in]kNumber of turns of each player.
[in]pNumber of player for whom initialization is called.
[in]x1Column number of first player's king.
[in]y1Row number of first player's king.
[in]x2Column number of second player's king.
[in]y2Row number of second player's king.
[in,out]treeTree containing nodes with all existing pieces.

[out]kMyVariable keeping number of turns.
[in]p1InitializedInformation if INIT of the first player has already been called.

Zwraca
